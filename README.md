## Installation

Clone the repository with git:

```shell
git clone git@github.com:cgm42/WorkLand.git
```

Install dependencies in the client folder with [npm](https://npmjs.com):

```shell
npm run workland
```

Install dependencies in the server folder:

```shell
npm i
```
